# Practica-3_-Edwin
Este repositorio contiene la solución a una práctica de Análisis Exploratorio de Datos (EDA) avanzado, centrada en la creación y aplicación de una librería de visualización personalizada en Python.  El objetivo principal fue profundizar en las técnicas de preprocesamiento de datos y generar visualizaciones estadísticas complejas y reutilizables.
